## [0.1.0a1] - 2024-12-22

### Added

- Initial alpha release
- Core reactivity system (Signal, Effect, Computed)
- UI components (Flex, Box, Text, Button, Input)
- Web renderer with PyFuseByte compiler
- TUI renderer with Flexbox layout engine
- CLI commands: `pyfuse dev`, `pyfuse build`, `pyfuse new`, `pyfuse learn`
- PyPI distribution as `pypyfuse` (import as `pyfuse`)
