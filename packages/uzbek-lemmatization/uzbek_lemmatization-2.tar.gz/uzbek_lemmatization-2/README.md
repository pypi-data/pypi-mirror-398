# Lemma & Lemmatization
There are 4 different ways of lemmatization: **rule**, **dictionary**, **model**, **hybrid**.

The package finds lemmas of Uzbek words based on the dictionary.

# Install

```bash
pip install uzbek_lemmatization
```
