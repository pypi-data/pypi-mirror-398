# Примеры использования

Эта папка содержит пример рабочего приложения для управления отелем, созданного на основе шаблонов из `pyqt_template_app`.

## Файлы

- `main.py` - точка входа приложения
- `db.py` - модуль работы с базой данных
- `login_window.py` - окно авторизации
- `admin_window.py` - окно администратора
- `manager_window.py` - окно руководителя
- `client_window.py` - окно клиента

## Использование

Этот пример демонстрирует, как использовать шаблоны для создания реального приложения.

```python
# Пример импорта из установленного пакета
from pyqt_template_app.examples import main
# или
import pyqt_template_app.examples.main as app
app.run()
```

## Структура БД

Для работы примера требуется создать базу данных MySQL по структуре из `БД.txt` (корень проекта).
