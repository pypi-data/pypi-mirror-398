## Usage

### Installation

To install the `flet-ads` package and add it to your project dependencies:

- Using `uv`:
    ```bash
    uv add flet-ads
    ```

- Using `pip`:
    ```bash
    pip install flet-ads
    ```
    After this, you will have to manually add this package to your `requirements.txt` or `pyproject.toml`.

- Using `poetry`:
    ```bash
    poetry add flet-ads
    ```
